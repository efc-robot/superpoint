# SuperPoint Library
用于ORB_SLAM2_SP，也可以单独运行。
## 安装
- 先安装caffe-ssd，https://github.com/zoeyuchao/caffe-ssd.git
- 安装superpoint

 `git clone https://github.com/zoeyuchao/superpoint.git`
    
 `cd superpoint && mkdir build`

 `cd build && cmake .. && make`

## 使用
  提供了一个example演示使用方法。

  `cd example && ./example`
